## 🐍 Manipulação de Listas em Python
As listas são coleções ordenadas, mutáveis e que permitem elementos duplicados. No contexto de análise de dados, elas são frequentemente usadas para armazenar nomes de colunas, filtros ou resultados de loops.

1. Ordem e Posição (Índices)
Cada item em uma lista tem um "endereço" numérico.

O primeiro item sempre ocupa a posição 0.

Você pode acessar itens do fim para o começo usando contagem regressiva.

Isso permite que você saiba exatamente onde cada informação está guardada.

2. Dinamismo e Flexibilidade
Diferente de uma pasta de arquivos física que tem um tamanho fixo, as listas em Python são "elásticas":

Adição: Você pode colocar um novo item no final da fila ou "furar a fila" inserindo-o em qualquer lugar.

Remoção: É possível retirar um item sabendo o nome dele ou simplesmente apontando para a posição que ele ocupa.

Substituição: Você pode trocar o conteúdo de uma posição sem afetar o restante da lista.
